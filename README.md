# C.A.R.S. Senior Design
## Rapid Tower Deployer

### Pinouts:
#### GPS (Start from left):
1. Empty
2. TX -> 1
3. RX -> 0
4. GND -> GND
5. Vcc -> 5v

#### Pressure Sensor:
1. Vout -> A0
2. GND -> GND
3. Vcc -> 5v

#### Relay Module:
1. GND -> GND
2. IN1 -> 9
3. IN2 -> 8
4. IN3 -> 7
4. IN4 -> 6
5. Vcc -> 5v

#### ToF Sensor (Pin 1 starts on brown wire):
1. SCL -> SCL (17)
2. SDA -> SDA (16)
3. GND_LED -> GND
4. Vcc_LED -> 5v
5. Empty
6. Empty
7. GND -> GND
8. Vcc -> 5v
